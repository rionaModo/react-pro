## softapp 是基于node、express、vue的单页面应用，采用mongodb作为数据库，是一个完整的具有操作业务的工程
## 配置文件路径：
server/config/default.js

## 端口配置  ：默认9051
在配置文件中将端口port 的值进行修改

# clinet ：
## vue模板路径
client/src/views/components
## 路由配置文件路径
client/router.js

操作：
## 1.安装依赖
 npm install
## 2.启动工程
npm start

访问例子：http://localhost:9051/foo
